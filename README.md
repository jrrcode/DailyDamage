# Daily Damage

Daily Damage is a personal finance tracker built with React and Vite.

## Features

- Dashboard summaries for money, expenses, recurring bills, credit, and loans
- Money accounts grouped by banks, e-wallets, and credit cards
- Expense tracking with recurring payment confirmation
- Credit and loan tracking with due dates and payment schedules
- Dark and light themes

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
