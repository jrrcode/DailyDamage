The app first looks for transparent local PNG logo files in this folder. If a PNG is missing, it falls back to a remote brand icon.

Use transparent PNGs with these exact filenames:

- bpi.png
- gcash.png
- maya.png
- bdo.png
- unionbank.png
- shopee.png
- lazada.png

If both the local PNG and remote icon cannot load, the app falls back to the account type icon.
